# Mini_Pixel
## Написание кода
Код для МК ESP32-S3, код можно писать в Visual Studio Code с дополнением PlatformIO, на данный момент тестируется и будет проверятся
Не забываем загрузжать всё на GitHub, делать через VS Code можно из коробки, главное иметь Git на компьютере, и настроить чуток(https://habr.com/ru/articles/541258/).
