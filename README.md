# **Mini_Pixel**
## Написание кода
Код для МК ESP32-S3, код можно писать в Visual Studio Code с дополнением PlatformIO, на данный момент тестируется и будет проверятся.
Не забываем загрузжать всё на GitHub, делать это через VS Code можно из коробки, главное иметь Git на компьютере, и настроить чуток (https://habr.com/ru/articles/541258/).

## Тестирование логической схемы
Будет проводится в Proteus
|№|Элементы|Ссылка на документацию| Коментарии |
|-|-|-|-|
|1.| ESP32-S3 |||
|2.| DC CONVERTER LM2596 |||
|3.| SONGLE SRD-05VDC-SL-C |||
|4.| ili9488 x2|| В Protues можно использовать аналог|
|5.| Шаговый двигатель nema17 17mm |||

№	Компонент	Назначение	Интерфейс / Питание	Техническое примечание
1	ESP32-S3	Центральный контроллер	3.3V DC / Wi-Fi & BLE	Основной вычислительный модуль с поддержкой AI-инструкций.
2	LM2596	Понижающий DC-DC преобразователь	Вход: 7-35V / Выход: 5V	Обеспечивает стабильное питание логики и катушки реле.
3	Songle SRD-05VDC	Электромагнитное реле	5V (Coil) / 250V AC (Switch)	Требуется транзисторный ключ (NPN/MOSFET) и защитный диод.
4	ILI9488 (x2)	TFT-дисплеи (480x320)	SPI Interface / 3.3V	Подключение через общую шину SPI с разделением сигналов CS.
5	Nema 17 (17mm)	Шаговый двигатель	12V-24V (через драйвер)	Компактная версия (pancake). Требуется драйвер типа A4988 или TMC2209.
